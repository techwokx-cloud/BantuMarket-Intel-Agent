## 🚀 QUICK START

### 1. Set Up Environment Variables

```bash
# Copy template
cp .env.example .env

# Edit with your credentials
nano .env  # or your favorite editor
```

### 2. Add Your API Keys

```bash
# Get from https://console.anthropic.com/api_keys
ANTHROPIC_API_KEY=sk-ant-xxxxxx

# Get from https://brightdata.com/cp/zones  
BRIGHT_DATA_API_KEY=your_key_here
BRIGHT_DATA_SERP_ZONE=serp_api1
BRIGHT_DATA_CUSTOMER_ID=hl_XXXXXX
```

### 3. Test Locally

```bash
pip install -r requirements.txt
streamlit run bantumarket_finance_app.py
```

### 4. Deploy

```bash
# Push to GitHub
git add .
git commit -m "BantuMarket Track 2 - Finance Intelligence"
git push origin main

# Deploy on Streamlit Cloud
# Go to streamlit.io/cloud
# New app → Select repo → bantumarket_finance_app.py
# Add Secrets in Settings → Secrets tab
```

## 📍 WHERE TO CREATE .env FILE

### Local Development
```
your-project/
├── .env                          ← Create HERE
├── .env.example                  ← Template (reference only)
├── bantumarket_finance_app.py
├── requirements.txt
└── ...
```

### Streamlit Cloud
**Do NOT upload .env file**

Instead:
1. Deploy your repo to Streamlit Cloud
2. Click Settings → Secrets tab
3. Add secrets one by one:
   ```
   ANTHROPIC_API_KEY=sk-ant-xxxxx
   BRIGHT_DATA_API_KEY=xxxxx
   BRIGHT_DATA_SERP_ZONE=serp_api1
   BRIGHT_DATA_CUSTOMER_ID=hl_xxxxx
   ```
